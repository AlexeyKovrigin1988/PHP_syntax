Создайте интерфейс логирования LoggerInterface. Пусть данный интерфейс содержит следующие методы:
logMessage — записать сообщение в лог. Метод получает на вход текст ошибки.
lastMessages — получить список последних сообщений из лога. Метод принимает на вход количество сообщений, которые нужно получить, и возвращает массив сообщений. 
Создайте интерфейс для обработки событий EventListenerInterface. Этот интерфейс должен содержать следующие методы:
attachEvent — присвоить событию обработчик. Принимает на вход имя метода класса, при выполнении которого он должен быть вызван, и колбек-функцию, которая вызывается при возникновении события.
detouchEvent — убрать обработчик события. Принимает на вход имя метода класса, при выполнении которого вызывается обработчик события.
Измените класс Storage таким образом, чтобы он реализовывал оба интерфейса: LoggerInterface и EventListenerInterface.
Измените класс User таким образом, чтобы он реализовывал интерфейс EventListenerInterface.
