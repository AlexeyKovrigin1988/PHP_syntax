Задание 1
Дана строка
«Лето пришло».
Присвойте указанную фразу в виде значения переменной.
Найдите в этой фразе пробел (с помощью функции поиска подстроки в строке).
Присвойте полученное значение новой переменной.
Выведите полученную переменную на экран.
Присвойте ещё одной новой переменной часть строки после пробела (с помощью функции разбиения строк на части и созданной выше переменной).
Выведите на экран последнюю переменную.

Задание 2
Дана строка с повторными словами:
«Я люблю море. Я лечу на море. Я умею плавать в море. Какое чистое море! Хочу на море. Завтра поедем на море.»
Присвойте указанную фразу в виде значения переменной.
Найдите в ней повторяющееся слово и присвойте его в виде строки в другую переменную.
Замените повторные слова в указанной фразе на те же самые (с помощью функции для замены подстрок в строках), перевёрнутые в обратном порядке с помощью функции strrev.
Условие: в функции замены подстрок нужно использовать созданные переменные.

Задание 3
Создайте переменную с ассоциативным массивом из трёх элементов, где ключами будут латинские буквы a, b и c, а значениями — любые строки на ваше усмотрение.
Присвойте новой переменной значение в виде простого массива, являющегося набором ключей массива из первой переменной.
Подсказка: используйте одну из функций разбиения массива на ключи/значения.
И создайте третью переменную, значением которой должна быть строка, состоящая из значений второй (простого массива), разделённых пробелами.
Выведите на экран значение третьей переменной с помощью конструкции echo.

Задание 4
Создайте переменную, в которую положите ассоциативный массив из пяти элементов. Причём ключи должны быть уникальные, а как минимум два значения должны повторяться и идти НЕ подряд.
Посчитайте количество элементов в массиве с помощью функции определения размера массива и присвойте результат работы этой функции в новую переменную. Выведите её на экран.
Переверните массив с помощью функции замены ключей на значения, а значений на ключи и положите результат в ещё одну переменную.
Теперь снова определите размер полученного массива и положите это значение в очередную переменную. Также выведите её на экран.
Последним шагом выведите на экран результат сравнения переменной с размером первого массива и переменной с размером второго массива.

Задание 5
Скопируйте код предыдущего задания в новый файл с этим заданием.
Возьмите значения массива из пункта 3 предыдущего задания с помощью одной из функций разбиения массива на ключи/значения и положите результат в новую переменную.
Добавьте к полученному массиву дополнительный простой массив из одного (!) элемента, которого нет среди уже существующих элементов, а результат сохраните в новую переменную.
Подсказка: используйте функцию объединения массивов.
Теперь объедините полученный простой массив на предыдущем шаге со списком значений массива, созданного в пункте 1 предыдущего задания.
Полученный объединённый массив присвойте переменной и выведите её на экран.
